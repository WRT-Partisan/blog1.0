前言

标准模板库 STL（Standard Template Library），是 C++ 标准库的一部分，不需要单独安装，只需要#include 头文件。

C++ 对模板（Template）支持得很好，STL 就是借助模板把常用的数据结构及其算法都实现了一遍，并且做到了数据结构和算法的分离。

STL六大部件

  • 容器（Containers）
  • 分配器（Allocators）
  • 算法（Algorithm）
  • 迭代器（Iterators）
  • 适配器（Adapters）
  • 仿函数（Functors）

要真正提高C++编程效率，需要将STL六大部件结合使用，才能大放异彩。

容器

vector

可变长度数组。

构造

一维数组

基本方法：vector<类型> 数组名(长度,[初值])（括号内的内容可以不写，默认长度为）。

构造数组，并用初始化：vector<类型> 数组名{a,b,c,...};。

时间复杂度：。

二维数组

构造指定行数，不指定列数的二维数组：vector<类型> 数组名[行];（常用于邻接表存图）。

构造二维数组基本方法：vector< vector<类型> > 数组名;。

构造指定行、列、初值的二维数组：vector< vector<类型> > 数组名(行,vector<类型> (列,初值))。

用法作用

数组.clear()清空数组
数组.empty()如果数组是空返回 true，反之返回 false
数组.resize(新长度,[默认值])修改数组的长度。如果是缩短，则删除多余的值；如果是扩大，且指定了默认值，则新元素均为默认值（旧元素不变）
数组.begin()指向第一个元素的迭代器
数组.end()指向最后一个元素的下一个位置的迭代器
数组.rbegin()反向迭代器，指向最后一个元素
数组.rend()反向迭代器，指向第一个元素之前的位置
数组.front()返回数组的首元素
数组.back()返回数组的尾元素

算法

swap

交换两个相同类型数据的值。

void swap( T& a, T& b );。

sort

快速排序。

void sort( RandomIt first, RandomIt last, Compare comp );。

在中进行排序。

比较器默认为 less<类型>（从小到大），从大到小为 greater<类型>。

自定义排序

自定义comp函数。

bool comp(类型 前一个位置元素,类型 后一个位置元素);。

comp函数的返回值是一个bool值，当返回值为true时不改变元素顺序，反之则需要调换元素（可换可不换时要交换）。

使用方法

用法作用
sort(数组,数组+n)对数组到排序
sort(数组+1,数组+n+1)对数组到排序
传入迭代器对 vector、字符串等进行排序

lower_bound/upper_bound

二分查找。

在已升序排序的元素中，应用二分查找检索指定元素，返回对应元素迭代器位置。找不到则返回尾迭代器。

lower_bound(_ForwardIterator __first, _ForwardIterator __last,const _Tp& __val, _Compare __comp)

upper_bound(_ForwardIterator __first, _ForwardIterator __last,const _Tp& __val, _Compare __comp)

lower_bound()：寻找的第一个元素的位置。

upper_bound()：寻找的第一个元素的位置。

返回的是迭代器，减去头迭代器即可转成下标索引。

比较器默认为 less<类型>（从小到大），若序列从大到小为 greater<类型>。

自定义比较

自定义comp函数。

bool comp(类型 前一个位置元素,类型 后一个位置元素);。

comp函数的返回值是一个bool值，当返回值为true时往左找，反之向右找。

reverse

void reverse( BidirIt first, BidirIt last );

反转一个可迭代对象的元素顺序。

max/min

返回最大值/最小值的数值。

template< class T > const T& max( const T& a, const T& b );。

template <class T> const T& min (const T& a, const T& b);。

两个元素

max(第一个元素,第二个元素)。

min(第一个元素,第二个元素)。

多个元素

使用列表构造语法传入一个列表，这样就能一次性给多个元素找最值。

max({第一个元素,第二个元素,第三个元素,...})。

min({第一个元素,第二个元素,第三个元素,...})。

unique

消除数组的重复相邻元素，数组长度不变，但是有效数据缩短，返回的是有效数据位置的结尾迭代器。

ForwardIt unique( ForwardIt first, ForwardIt last );。

单独使用 unique 并不能达成去重效果，因为它只消除相邻的重复元素。但是如果序列有序，那么它就能去重了。

去重 vector 时序列尾部会产生一些无效数据，为了删掉这些无效数据，我们需要结合 erase。

数组.erase(unique(数组.begin(),数组.end()),数组.end())。

所有函数参数均支持int/long long/float/double/long double。

公式示例

注意事项

由于浮点误差，有些的数学函数的行为可能与预期不符。

    • 别用：floor(1.0 * a / b)。
    • 要用：a / b。
    • 别用：ceil(1.0 * a / b)。
    • 要用：(a + b - 1) / b。（）
    • 别用：(int) sqrt(a)。
    • 要用：二分查找。
    • 别用：pow(a, b)。
    • 要用：快速幂。
    • 别用：log2(a)。
    • 要用：__lg/bit_width。
